Oracle Loading Instructions

1. Log in to Oracle Fusion with data migration privileges
2. Navigate to: Tools → Scheduled Processes → Schedule New Process
3. Search for: Load Interface File for Import
4. Upload the FBDI CSV, select the correct Import Process name
5. Submit and monitor in Scheduled Processes
6. Review errors: Tools → Import Management → Import Queue
